# API Testing

## Overview

This folder contains API testing documentation created using Postman.

API testing validates the backend services of the CloudVault application by verifying request methods, response codes, authentication, and JSON responses.

## API Operations

- Login API
- Registration API
- File Upload API
- File Download API
- Rename API
- Delete API
- Search API
- Profile API
- Logout API

## Tools Used

- Postman
- REST API
