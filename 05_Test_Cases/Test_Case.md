# Test Cases

## Overview

This folder contains detailed manual test cases for all functional modules of the CloudVault application.

Each test case includes:

- Test Case ID
- Requirement ID
- Scenario ID
- Preconditions
- Test Data
- Test Steps
- Expected Result
- Actual Result
- Status
- Priority

## Modules

- Login
- Registration
- Dashboard
- File Upload
- File Download
- Rename File
- Delete File
- Search
- Profile
- Logout
