
# ☁️ CloudVault-QA

A complete Manual Testing portfolio project demonstrating the Software Testing Life Cycle (STLC) for a cloud storage web application.




# 📌 Project Overview

CloudVault is a secure cloud storage web application that allows registered users to securely manage their files online.

Users can:

* Register
* Login
* Upload Files
* Download Files
* Rename Files
* Delete Files
* Search Files
* Create Folders
* Update Profile
* Logout



# 🎯 Project Objective

The objective of this project is to perform end-to-end Manual Testing by preparing all QA deliverables required during the Software Testing Life Cycle (STLC).

This repository demonstrates requirement analysis, test planning, test design, bug reporting, API testing, database validation, and final test reporting.
